Программа по реализации калькулятора по шаблону 8, сделано с помощью Dear ImGui.

Сборка:
1) Скачать исходники, убедиться что все зависимости (а именно библиотека Dear ImGui) находятся в нужных директориях. Так-же, программа создана для ОС Windows (разработана и проверена на Windows 11), а так-же требует видеокарту с совместимостью с DirectX11.
2) Открыть проект в Visual Studio 2022
3) Выбрать решение Calculator.sln
4) Собрать решение через главное меню Visual Studio: Build->Build Solution.
